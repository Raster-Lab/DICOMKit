# DICOMToolbox Keyboard Shortcuts

Quick reference for all keyboard shortcuts in DICOMToolbox. Shortcuts use standard macOS modifier key notation:

| Symbol | Key          |
|--------|--------------|
| ⌘      | Command      |
| ⌥      | Option       |
| ⇧      | Shift        |
| ⌃      | Control      |
| ↩      | Return/Enter |
| ⌫      | Delete       |
| ⎋      | Escape       |

All shortcuts can be customized in **DICOMToolbox → Settings… → Shortcuts**.

## Accessibility

DICOMToolbox supports full keyboard navigation following macOS accessibility standards.

| Shortcut       | Action                                    |
|----------------|-------------------------------------------|
| Tab            | Move focus to next control                |
| ⇧Tab           | Move focus to previous control            |
| Space          | Activate focused button or toggle         |
| ↩              | Confirm focused action                    |
| ⌘⇧+            | Increase console font size                |
| ⌘⇧-            | Decrease console font size                |
| ⌘⌥I            | Toggle VoiceOver descriptions             |

DICOMToolbox also supports:

- **Dynamic Type** — Respects system text size preferences.
- **Reduced Motion** — Disables animations when the system preference is enabled.
- **High Contrast** — Increases contrast for UI elements in high contrast mode.
- **Full VoiceOver support** — All controls and output regions have descriptive labels.
